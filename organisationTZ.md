# Организатор

### 2. Структура профиля

| Зона | Назначение | Кто видит |
| :-- | :-- | :-- |
| Личные данные | Конфиденциальная информация (email, телефон, пароль, настройки) | Только владелец |
| Информация об организации | Данные организации, которую представляет организатор (название, тип, контакты, логотип) | Все (публично) |
| Управление мероприятиями | Создание, редактирование, публикация мероприятий | Только владелец |
| Управление участниками | Загрузка результатов, подтверждение достижений, коммуникация с участниками | Только владелец |
| Аналитика и отчетность | Статистика по мероприятиям, вовлеченность участников, популярность | Только владелец |
| Публичная витрина | Список мероприятий организатора, информация об организации | Все |

### 5. Управление мероприятиями

| ID | Требование | Описание | Приоритет |
| :-- | :-- | :-- | :-- |
| PROF-O-21 | Кнопка «Создать мероприятие» | Доступна только владельцу | must |
| PROF-O-22 | Форма создания | Поля: название, тип (олимпиада / конференция / хакатон / курс), уровень (международный / всероссийский / региональный / вузовский / школьный), даты (начало, окончание), формат (очно / онлайн / гибрид), местоположение (если очно), описание, ссылка на сайт мероприятия (если есть), контактный email организаторов, логотип/баннер мероприятия | must |

### 11. Структура данных (полный перечень полей)

#### Публичные поля организатора

* `id` — UUID
* `logoUrl` — string
* `organizationName` — string
* `shortName` — string
* `organizationType` — enum (university, scientific, olympiad, conference, foundation, educational, other)
* `website` — string
* `description` — text
* `contactEmail` — string (публичный)
* `contactPhone` — string (опционально)
* `socialLinks` — object { telegram, vk, youtube, other[] }
* `foundedYear` — integer
* `eventsCount` — integer (рассчитываемый)
* `totalParticipants` — integer (рассчитываемый)
  

#### Личные / приватные поля

* `email` — string (личный, уникальный)
* `phone` — string (личный, опционально)
* `passwordHash` — string
* `notificationSettings` — object { verificationRequests, newRegistrations, reports, deliveryChannels }
* `createdAt` — datetime
* `lastLoginAt` — datetime
  


#### Мероприятие (объект)

* `id` — UUID
* `organizerId` — UUID
* `title` — string
* `type` — enum (olympiad, conference, hackathon, course, volunteering, other)
* `level` — enum (international, national, regional, university, school)
* `dates` — object { start, end, registrationDeadline }
* `format` — enum (offline, online, hybrid)
* `location` — string (если offline)
* `description` — text
* `website` — string
* `contactEmail` — string
* `logoUrl` — string
* `bannerUrl` — string
* `qrCodeUrl` — string
* `status` — enum (draft, published, completed, cancelled)
* `participantsCount` — integer (рассчитываемый)
* `verifiedAchievementsCount` — integer (рассчитываемый)
* `customFields` — array (кастомные поля регистрации)
* `createdAt` — datetime
